# Weather-app-API-
A weather application built with JavaScript that fetches real-time weather data using a public weather API. The app displays current weather conditions, temperature, humidity, and forecasts based on user location or search queries.

## Features
- Real-time weather data
- Search by city name
- Displays temperature, humidity, and wind speed
- Weather icons and descriptions
- Responsive design

## Technologies
- HTML5
- CSS3
- JavaScript (ES6+)
- Weather API

## Installation
1. Clone the repository
2. Open `index.html` in your browser
3. Allow location access or search for a city

## Usage
Enter a city name to view current weather conditions and forecast information.
